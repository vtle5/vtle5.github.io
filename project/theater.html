Movie Theaters and Pandemics

It was a month into the Covid-19 pandemic and businesses were adapting to the new changes that were being put in place. Movie theaters in particular had to adapt social distancing protocols while still attaining optimal seating. My class was given several methods to come up with a solution to this problem. One of them involving genetic algorithms.

Genetic Algorithms

I have heard of genetic algorithms before and had understood how they functioned at the time. However, this would be a first for creating one. Doing some research had lead me to a better understanding of how these algorithms functioned. The program turned out to be functional without much issue. But there was a flaw; the algorithm was not learning effectively. So the project became a matter of determining how each individual in the algorithm should be scored. Eventually a satisfactory scoring system was made and as a result the seating models had improved with each generation.
